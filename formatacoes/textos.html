Formatando textos


Tags comuns

Negrito - tag <b>
Strong (negrito) - tag <strong> - dá mais ênfase ao texto do que a tag <b> - melhora a acessibilidade
Itálico - tag <i>
Sublinado - tag <u>
Marca texto - tag <mark>
Sobrescrito - tag <sup>
Subescrito - tag <sub>
Blockquote - dá um espaçamento no texto - tag <blockquote>

Pequeno - tag <small>
Texto deletado - tag <del>
Texto inserido - tag <ins>
Código - tag <code>
Preformatted - tag <pre>

Definição - tag <dfn>
Sigla - tag <abbr>
Autor - tag <cite>
Data - tag <ime>
Variável - tag <var>
Amostra - tag <samp>

Tag <font>

Usando a tag <font> para alterar a formatação do texto. Não é muito utilizada pois o CSS faz isso.

Atributos mais comuns

  • colorcor
  • sizetamanho
  • face fonte - pode deixar mais opçoes de fonte, para o navegador poder buscar no computador da pessoa.
