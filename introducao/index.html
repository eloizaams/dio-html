Tags i, u e strong

(voltar)

A tag "i" serve para deixar o texte em italico, a tag "u" para dexar sublinhado e a "strong" para deixar em negrito.

Tags ol, ul e li

(voltar)

A tag "ol" é utilizada para estruturar uma lista ordenada e a "ul" uma lista não ordenada. A tag "li" indica os itens da lista.

Tag a

(voltar)

A tag "a" é usada em links

  • Uso: Cria um link clicável que leva o usuário para outra página, arquivo ou recurso online.
  • Atributos importantes:
    • href: Define a URL do destino do link
    • target: Define onde o link será aberto (mesma página, nova aba, etc).
