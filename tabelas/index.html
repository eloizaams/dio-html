Uso de tabelas em HTML

Usda para apresentar dados em formato tabular.

Tag <table>

Essa tag define o espaço do HTML onde serão trabalhados os dados tabulares.

Atributos da tag <table>

  • border
  • width
  • align
  • cellspacing
  • cellpadding
  • sumary

Tag <tr>

tr é usado para delimitar uma linha da tabela, engloba todas as células que compõe a linha.

Efeito "zebrado" à tabela pode-se usar o estilo :nth-child(even) no CSS.

Para que a linha fique em evidência quando passa o mouse usa-se o estilo :hover td no CSS,

Tag <td> e <th>

  • th indica o cabeçalho da tabela. Deve-se colocar o menor nome possível para o campo e usar o atribulto title para colocar uma descrição mais completa que vai aparecer quando passar o mouse sobre o campo.
  • th indica as células que compõem a linha da tabela

Tags <tbody>, <thead> e <tfoot>

  • tbody tag usada para envolver o corpo da tabela.
  • thead tag usada para envolver o cabeçalho da tabela.
  • tfoot tag usada para envolver o rodapé da tabela.

Tag <caption>

Contém a descrição da tabela. É o nome da tabela. Aparece no topo da tabela.

Box Shadow

Site que facilita a geração do código para a sombra a ser aplicada à tabela.

Box Shadow Generator

Mesclar células

Para mesclar células dentro de uma linha da tabela, utiliza-se o atributo colspan e coloca como valor quantas células devem ser mescladas. Para mesclar colunas utiliza-se o atributo rowspan e coloca como valor quantas linhas devem ser mescladas.
